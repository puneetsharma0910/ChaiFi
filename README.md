# Chaifi ☕  

Chaifi is a Patreon-style crowdfunding platform where fans can support creators by **buying them a chai**.  

## 🚀 Features  
- Fans can support creators with small donations  
- Clean and responsive UI with Next.js and Tailwind CSS  
- More features coming soon...  

## 🛠️ Tech Stack  
- **Frontend:** Next.js, React, Tailwind CSS  
- **Backend & Authentication:** (Coming soon)  

## ⚡ Getting Started  
```bash
git clone https://github.com/yourusername/chaifi.git  
cd chaifi  
npm install  
npm run dev
